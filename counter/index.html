操作説明

各種ボタン

「背景」ボタンから一枚絵をアップロード
「ZIP」ボタンからパネル開け画像生成ツールで生成したZIPファイルをアップロード
「保存」ボタンで進捗画像をダウンロード(画質が下がったものがダウンロードされます)
企画が終了したら「データ消去」ボタンを押すようにして下さい(スマホの容量が少し軽くなります)

文字入れ

ZIPファイルをアップロードするとパネル毎に文字が入れられます
パネル名の横をタップするとレイヤー選択状態になるので、文字を入力して下さい
カラー選択は左が文字、右が文字枠に対応しています

スマホ操作

1本指ドラッグ → 移動
2本指ピンチ → サイズ変更
2本指回転 → 回転

PC操作

ドラッグ → 移動
ホイール → サイズ変更
Alt + ホイール → 回転

文字が表示されない時はレイヤーを変更してみてください
